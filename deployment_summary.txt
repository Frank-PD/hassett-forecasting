--------------------------------------------------------------------------------
DEPLOYMENT INSTRUCTIONS
--------------------------------------------------------------------------------

1. Load the routing table: route_model_routing_table.csv

2. For each route to forecast:
   a. Look up route in routing table by:
      - route_key (exact match)
      OR
      - ODC + DDC + ProductType + dayofweek

   b. Use the 'Optimal_Model' specified for that route

   c. Check 'Confidence' level:
      - HIGH: Error typically < 20%
      - MEDIUM: Error typically 20-50%
      - LOW: Error typically > 50% (use with caution)

3. If route not found in routing table:
   - Use default model: 04_Recent_8W
   - This is the best performing single model overall
